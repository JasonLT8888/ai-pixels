# 待办

本文档汇总当前项目里已经确认缺失、但值得后续补齐的功能。优先级基于当前产品闭环、旧文档承诺和实现成本综合判断。

## P1

- [ ] 左侧绘图工具栏
  - 现状：左侧仍为占位区域
  - 最小目标：提供选择工具、基础颜色/模式入口，为后续交互式绘制留接口

- [ ] 画布交互式绘制
  - 现状：画布只支持缩放和平移
  - 最小目标：支持点击/拖拽生成基础指令，如单点、直线、矩形

- [ ] 指令列表可视化编辑
  - 现状：只能通过 JSON 文本整体编辑
  - 最小目标：支持单条指令的增删改、排序与局部应用

## P2

- [ ] 项目复制 / fork
  - 价值：便于在现有项目基础上分支尝试不同方向

- [ ] 单条消息重发与编辑
  - 价值：多轮对话里纠错更高效，不必整段重发

- [ ] chat 搜索 / 筛选
  - 价值：chat 增多后更容易定位目标会话

- [ ] 指令差异对比
  - 价值：便于查看 AI 新回复与当前项目的差异

- [ ] 导出会话快照
  - 价值：便于问题复现、调试和分享提示词上下文

- [ ] 模型耗时与响应元信息展示
  - 价值：帮助评估模型稳定性、成本与交互体验

- [ ] 记住每个项目或 chat 的缩放与视口位置
  - 价值：回到项目时减少重复调整视图



## 建议实现顺序

- 左侧绘图工具栏最小版本
- 画布交互式绘制
- 指令列表可视化编辑
